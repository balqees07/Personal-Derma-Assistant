# Personal Derma Assistant

AI-powered skin condition analysis and personalized skincare recommendations using Deep Learning.

## Overview
Personal Derma Assistant is an AI-powered mobile application that analyzes facial skin conditions from a single image and provides personalized skincare recommendations and a daily skincare routine.

## AI Stack
- MobileNetV2 with Transfer Learning
- TensorFlow / Keras
- FastAPI
- Flutter
- Firebase
- OpenCV
- NumPy

## Dataset
- Kaggle dataset (~4,788 images)
- 9 skin condition classes

## Performance
- Training Accuracy: 90%
- Validation Accuracy: 82%
- Test Accuracy: 80.70%
- Macro F1-Score: 0.81

## Status
Working Prototype tested by 100+ users.
